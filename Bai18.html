Bài 2:Thay đổi định dạng và màu sắc của phông chữ trong các vùng khi di chuyển chuột qua.

Giải:

.name {
/* Thay đổi màu chữ khi di chuột qua */
.nav a:hover {
color: #ff5722;
}

/* Thay đổi màu nền và viền khi di chuột qua các nút */
button:hover {
background-color: #ff5722;
border: 1px solid #e64a19;
}
}

Giải:

.name {
                  /* Định dạng banner */

}

.slogan {
